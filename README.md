# Projectgraph CLI

## Install

Make sure you have .NET Core SDK 3.0 or higher installed on your machine. Then run

```bash
dotnet tool install -g DotNet.ProjectGraph.Tool
```

## Run

```bash
projectgraph --help
```

## Develop

### Prerequisites

* VS Code
* VS 2019 or higher
* .NET Core 3.0 SDK or higher

### Get started

Open in VS Code, run the task `pack and install`.
